# 00 - 项目概览：简易博客系统全景

> 视频讲解稿 · 建议时长 5 分钟

---

## 开场白

大家好，今天我们来深入讲解一个完整的博客系统项目。这个项目虽然体量不大，但它覆盖了 Web 全栈开发中最核心的知识点：

- **后端**：Node.js + Express 框架，RESTful API 设计
- **数据库**：SQLite + SQL 语句，better-sqlite3 同步驱动
- **认证**：JWT（JSON Web Token）+ bcrypt 密码哈希
- **前端**：原生 HTML/CSS/JavaScript，不依赖任何框架

---

## 项目目录结构

```
boke/
├── frontend/                ← 前端：用户界面
│   ├── index.html           → 首页，展示所有文章
│   ├── login.html           → 登录页
│   ├── register.html        → 注册页
│   ├── create.html          → 写文章/编辑文章
│   ├── detail.html          → 文章详情，含编辑/删除入口
│   ├── my.html              → 「我的文章」管理列表
│   ├── css/
│   │   └── style.css        → 全局样式（响应式布局）
│   └── js/
│       ├── api.js           → 封装 fetch 请求
│       ├── auth.js          → 登录状态管理与导航栏渲染
│       └── main.js          → 防 XSS、日期格式化等工具函数
│
├── backend/                 ← 后端：API 服务
│   ├── server.js            → Express 应用入口，装配中间件与路由
│   ├── db.js                → SQLite 数据库连接 + 建表
│   ├── middleware/
│   │   └── auth.js          → JWT 认证中间件，保护受控路由
│   └── routes/
│       ├── auth.js          → POST /register + POST /login
│       └── posts.js         → GET/POST/PUT/DELETE 文章接口
│
├── docs/                    ← 文档（PRD + 本系列讲解稿）
├── package.json             → 项目元数据 + 依赖声明
└── node_modules/            → npm 安装的第三方包
```

## 数据流全景图

```
用户浏览器                     Node.js 服务器                  SQLite 数据库
─────────                    ──────────────                 ────────────
   │                              │                              │
   │  1. 打开 index.html          │                              │
   │ ──────────────────────────► │                              │
   │     (Express 返回静态页面)    │                              │
   │ ◄────────────────────────── │                              │
   │                              │                              │
   │  2. JS 发起 GET /api/posts   │                              │
   │ ──────────────────────────► │  SELECT * FROM posts          │
   │                              │ ──────────────────────────► │
   │                              │       返回数据行              │
   │                              │ ◄────────────────────────── │
   │      返回 JSON 数组          │                              │
   │ ◄────────────────────────── │                              │
   │                              │                              │
   │  3. 用户登录                  │                              │
   │  POST /api/auth/login        │                              │
   │ ──────────────────────────► │  SELECT FROM users            │
   │                              │ ──────────────────────────► │
   │                              │  bcrypt.compare(password)     │
   │                              │  jwt.sign({id, username})    │
   │     返回 JWT Token           │                              │
   │ ◄────────────────────────── │                              │
   │                              │                              │
   │  4. 发布文章（携带 Token）     │                              │
   │  POST /api/posts             │                              │
   │  Authorization: Bearer xxx   │                              │
   │ ──────────────────────────► │  jwt.verify(token)            │
   │                              │  INSERT INTO posts           │
   │                              │ ──────────────────────────► │
   │     返回 201 + postId        │                              │
   │ ◄────────────────────────── │                              │
```

## 核心技术栈一览

| 层面 | 技术 | 包名 | 作用 |
|------|------|------|------|
| 运行时 | Node.js | - | JavaScript 服务器运行环境 |
| Web 框架 | Express 4.x | `express` | 路由、中间件、静态文件 |
| 数据库 | SQLite | `better-sqlite3` | 嵌入式关系数据库，同步 API |
| 密码加密 | bcrypt | `bcryptjs` | 将密码哈希后存储，不可逆 |
| 令牌认证 | JWT | `jsonwebtoken` | 签发和验证身份令牌 |
| 跨域 | CORS | `cors` | 允许浏览器跨域请求（开发用） |
